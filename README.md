# Ambient AI Android Launcher

A battery-efficient, single-screen Android Launcher built with Jetpack Compose. Designed with a minimalist sci-fi aesthetic, this launcher replaces standard Android visual noise with a clean, edge-to-edge dashboard.

## Core Features
- **Edge-to-Edge UI:** Zero system bar padding for a true immersive experience.
- **Ambient AI Integration:** Persistent home screen chat powered by an Agentic AI system via Fireworks API.
- **Chrome PWA Setup:** Tailored to work cleanly alongside Chrome PWAs.
- **Dynamic Time-of-Day Theming:** Transitions between a "Calm Ocean" day theme and a "True Black OLED" night theme.
- **RSS & Notes Dashboard:** Built-in RSS feed reader and offline Room DB notes.
- **Squircle App Icons & Consistent Typography:** Strict design rules for zero visual noise.
- **Performance & Battery First:** Uses static pre-blurred PNGs and limits background processes.
