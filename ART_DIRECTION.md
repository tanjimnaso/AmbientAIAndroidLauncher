# Art Direction

## The Vibe
Calm ambient launcher meets editorial grouping, with a secondary industrial navigation layer.

The project is no longer aiming for a generic "sci-fi dashboard". The current visual language is:
- dark ocean wallpaper and scrims
- large rounded rectangles on home
- sparse grouping and deliberate whitespace
- a separate text-only, thin-line industrial page for structure and navigation

## Core Visual Rules
1. **Typography**
   - Use `DM Serif Display` for large moments and section emphasis.
   - Use `Inter` for body, labels, settings, and utility text.
   - Avoid mixed random weights and one-off display treatments.

2. **Color**
   - Home page stays ambient, cool, and wallpaper-led.
   - Late night should be deep ocean, not pure OLED black.
   - Small high-brightness accents are acceptable; large white surfaces are not.
   - The industrial page can be pale/paper-toned with thin black rules.

3. **Shapes**
   - Home uses large rounded card geometry.
   - Text-only rails and the industrial page use thin borders and flatter structure.
   - App icons can sit inside custom card geometry rather than relying on stock launcher icon spacing.

4. **Composition**
   - Group by intent, not by default Android launcher convention.
   - Break up app blocks with text-led rails and separators.
   - Browsers are allowed to be individual hero squares.
   - Wallet apps can be treated as a compact grouped strip.

5. **Navigation**
   - Home is scroll-first.
   - Left page is index-first.
   - Avoid a persistent bottom launcher bar.
