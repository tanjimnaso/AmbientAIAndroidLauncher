# Art Direction
*(To be discussed and expanded in detail later)*

## The Vibe
"Minimalist Sci-Fi Dashboard" meets "Ambient Calm".

## Key Directives
1. **Typography:**
   - Consistent, strict typography scaling (no mismatched text sizes, no visual noise).
   - Use a single primary font family (e.g., Orbitron or a clean geometric sans-serif) across all components.
2. **Color Palettes (Time-based):**
   - **Day Mode (Ocean):** Soft gradients, light ocean tones, dark grey text (no harsh blacks).
   - **Night Mode (Sci-Fi Dark):** True black (#000000) backgrounds for OLED battery saving, high-contrast accent text (Cyber Lime, Electric Blue).
3. **Shapes & Padding:**
   - Edge-to-edge elements with zero system padding constraints.
   - Squircle (superellipse) shapes for standard app icons.
4. **Backgrounds:**
   - NO real-time UI blur (Glassmorphism).
   - Use pre-blurred PNG static wallpapers for the background to maintain battery efficiency while achieving the ambient look.
